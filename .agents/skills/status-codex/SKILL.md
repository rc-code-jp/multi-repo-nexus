# Status Codex スキル

ChatGPT Codex の使用状況を確認します。

## When to Use

- Codex の使用率を確認したいとき
- ChatGPT Codex の usage ページを開きたいとき

## Instructions

以下のコマンドを実行して、ChatGPT Codex の使用状況ページを開く。

```zsh
open https://chatgpt.com/codex/settings/usage
```
